!File Menu (Ribbon)

UI Controls:
  • New Capture: initiates cross hairs so user can create a capturing region
  • Capture Regions Consecutively: Continues to prompt user with capturing region after each time they click the red button.
  • Restore Last Screen Capture: restores the size and position of the last captured region
  • Capture Last Capture: Automatically captures the last captured region
  • Capture Window: Hightlights sections of UI and when clicked it produces a capture
  • Capture Window with Auto-Scroll: method to capture an open window that is too long to be displayed in its entirety on the screen.
  • Capture Full Screen: method to capture everything (e.g., open applications, desktop icons and background, taskbar) displayed on your computer screen.
  • Capture Full Screen without Taskbar: method to capture everything (e.g., open applications, desktop icons and background, taskbar) displayed on your computer screen with the exception of the taskbar
  • Capture New Image from Clipboard: creates a new image based on what is currently in your Windows Clipboard
  • Capture Shape Area: method to capture an area covered by a shape that you have drawn.

Capturing Region

  • Insert Captured Region will invoke the targeting cursor shown above
  • By default,Capture should minimize during this step
  • The Pixel box shows the exact pixel location that the center of the targeting cursor is hovering over
  • Hitting ESC will return you to Capture, canceling the operation
  • A zoom box will appear in the upper left corner of your monitor,

 

Capturing Region Controls

 

  • The capture region box will have 8 resize points located around it
  • Each point can be used to resize the capture region
    • Middle boxes can be used to resize either left/right or up/down
    • Corner boxes can be used to resize the entire capture region
  • The region box will also have a cross-hair in center
    • Clicking and dragging this box will move the capture region accordingly
  • Verify resizing the capture region box will still result in an accurate capture
  • Three buttons will be displayed in the lower right corner

 

  • Capture Button: Click to capture what lies within the boundaries of the region
  • Width and Height: Can enter values to adjust the capturing region
  • Pick a Screenshot Size (Dropdown): Brings up the options context menu
  • Close Button/Cancel: Will cancel the operation and return the user to Capture

 

Capture Region Button Control

 

Pick a Screenshot Size

  • 320 * 240: A pre-defined size commonly used.
  • 640 * 480: A pre-defined size commonly used.
  • 800 * 600: A pre-defined size commonly used.
  • 1024 * 768: A pre-defined size commonly used.
  • Snap to a Window: Allows you to snap the region to an application window.
  • Move Window into Rectangle: Allows you to snap an application window into the region.

Recording Options

Options Context menu:

 

  • Snap to window... will open the Select Window dialog
  • Move Window into Rectangle... will open the Select Window dialog
  • Options... will open the options dialog

 

 

Snap to Window... brings up the Select Window dialog:

 

  • Displays all the currently opened windows
  • Selecting one and hitting OK will move the capture region box to the selected window
    • This will most likely change the size of the capture region box
  • Pressing Cancel will close the dialog and return you to capture region box

Move Window Into Rectangle... brings up the Select Window dialog:

 

  • Displays all the currently opened windows
  • Selecting one and hitting OK will move the selected window into the region box
    • This will most likely change the size of the selected window
  • Pressing Cancel will close the dialog and return you to capture region box
