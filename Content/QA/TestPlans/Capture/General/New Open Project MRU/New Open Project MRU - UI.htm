New Batch Job - Setup

 

 

UI Controls
  • Folder to Scan: Find and select the folder containing the images that you want to work with. Click OK.
  • Recursive If you want all images in subfolders to be scanned as well, click Recursive so that it contains a check mark.
  • Include Pattern: You can use the Include Pattern field to select the type of image files to be included in the scan. Click the down arrow and select the file type.
  • Profile to Apply: 6.Click in the Profile To Apply field and select the profile that you want to use. In addition to all of your custom profiles listed, you can also select "No Profile." This option is useful if you are a newly registered user and want to remove the trial-mode watermarks on all of your images.
  • Replace Shapes Option: If the profile contains shapes, you can make a selection in the Replace Shapes Option area to determine how those shapes are handled.
  • Put profile shapes beneath other shapes If the profile contains shapes, they will be added to the image. Any shapes currently on the image will be left alone.
  • Output Folder: You have the option of specifying an output folder where the images are saved as a result of the process. To do this, next to the Output Folder field, click Browse. Then find and select the folder containing the images that you want to work with, and click OK.

 

New Batch Job - Apply

 

 

UI Controls
  • Scan: The folder is scanned for images that match your settings. When it is finished, the image names are listed on the tab.
  • Apply Profile: The profile you selected is applied to all of the images.
  • Save File: The files are saved with the new profile settings. If you want to reuse the batch in the future, you can save it. To do this, press CTRL+S or select File>Saveto save your work. Then find a folder where you want to store the batch file, provide a name for it, and click Save.
