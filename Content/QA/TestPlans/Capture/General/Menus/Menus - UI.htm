Menus - Tool Strip UI

For the purposes of this test plan, verify all menu items function as they should and perform the Menu Test Cases for each menu. All menu items will be tested in more detail in the test plans of the modules which they support.

Accessing the Tool Strip (If Tool Strip is not enabled)

  • Go to !File | Options
  • In the "Interface" tab, select "Tool Strip"
  • Click OK

File Menu

Note: The Source Control Item will only show if your project is bound to a source control provider.

To access:

  • Click the File menu
  • Press [ALT] + F on the keyboard

File Menu Items (Tooltips in parentheses)

 

Menu Test Cases

  • Menu items are correct
  • Tooltips are correct
  • Spelling and grammar for items and tooltips are correct
  • No duplicate accelerators
  • All accelerators function
  • Keyboard can navigate and select options
  • Menu is easy to use and understand

Edit Menu

Note: The Source Control Item will only show if your project is bound to a source control provider.

To access:

  • Click the Edit menu
  • Press [ALT] + E on the keyboard

Edit Menu Items (Tooltips in parentheses)

 

  • Undo (Undo): Undoes the most recent action
  • Redo (Redo): Redoes the action that was most recently undone
  • Cut (Cut): Cuts the current selection to the clipboard
  • Copy (Copy): Copies the current selection to the clipboard
  • Paste (Paste): Paste the contents from the clipboard
  • Delete (Delete): Delete the current selection
  • Selection Rectangle: Select rectangular sections of the canvas
  • Magic Wand: Select irregular shaped sections of the canvas
  • Pencil: Draws on canvas
  • Eraser: Erases on canvas
  • Color Fill: Adds color to objects, selections, and the canvas
  • Crop Image: Invokes crop mode for full image.
  • Crop Selection: Invokes crop mode for image objects.
  • Discard History:Toggles Discard History on and off
  • Edit Text:Opens the text edit mode window for selected object
  • Select All (Select All): Selects all

File Menu Items (Tooltips in parentheses)

  • Object Layers: Open the Object Layers window
  • Palettes: Open the Palettes window

  • Start Page (Open the Start Page): Opens the Start Page
  • Messages (Open the Messages window): Opens the Messages pane
  • Status Bar: Hide\Show the Status Bar

View | Show Menu

View | Show Menu Items (Tooltips in parentheses)

  • Show Caret Tag Neighborhood (Toggle show Caret Tag Neighborhood): Turns the Caret Tag Neighborhood feature on or off.
  • Show Space (Toggle show space): Turns paragraph and space markers on and off in the active topic.
  • Show tags (Toggle show tags): Changes the XML Editor to "Show Tags" mode
  • Show Markers (Toggle show markers): Turns markers on or off in the active topic or other content file.
  • Show Inline Markers (Show markers for all inline elements): Lets you show or hide markers for all inline elements in the active topic.
  • Show invisible tags (Toggle show invisible tags): Turns invisible tags (e.g., <title>) on or off in the active topic. This option is enabled only if you have selected "Show tags."
  • Show Comments (Toggle show comments): Turns comment tags on or off in the active topic.
  • Show All Images as Thumbnails (Toggle show image thumbnails): Lets you toggle between displaying images as thumbnails or full size in the XML Editor.
  • Show tags in mono-color (): Lets you choose whether to display tags in a color or in black. This option is enabled only if you have selected "Show tags."
  • Show tags in mono-font (Toggle show tags in mono-font): Lets you choose whether to display all tags in the same font or in the font actually applied to each area. This option is enabled only if you have selected "Show tags."
  • Enable Object Positioning (Enable Object Positioning): Turns object positioning on and off.
  • Enable Orphan/Widow Control (Enable Orphan/Widow Control): Turns widow and orphan control on or off.
  • Show Hidden Elements (Toggle show hidden elements): Shows content that has been designated as "hidden." (the "display" property of the element is set to "none")
  • Show Conditional Indicators (Hide/show conditional indicators): Lets you show or hide conditional indicators in the XML Editor.
  • Show Variable Names (Toggle show variable names): Lets you show or hide variable names in your variable markers.
  • Highlight Variables (Highlight variables): Lets you choose whether to see a gray background behind variable definitions in a topic.
  • Show Bookmark Names (Toggle show bookmark names): Lets you show or hide the names of bookmarks in your markers (if you have inserted bookmarks) in the active topic.
  • Show Named Elements (Toggle show named elements): Lets you show or hide markers for all inline elements in the active topic.
  • Locks (Disabled/Enabled) (Switch to locked/unlocked mode): Locks or unlocks the content designated "locked"
  • Show Locked Elements (Toggle show locked elements): Allows you to show or hide the gray background associated with elements that have been tagged as "locked."
  • Show Changes (Toggle showing changes): Lets you show or hide the Annotations portion of the XML Editor.
  • Marker Width: Opens the Set Marker Width dialog.

  • Character: Shift+F11Invokes the Character sub-menu. This menu is covered in the character test plan.
  • Image… Ctrl+G (Insert an image): Invokes the Insert Picture dialog

    • Windows Media Player… (Insert an embedded Windows Media Player): Invokes the Insert Multimedia dialog with the file filter set to Windows Media Player file types.
    • Quicktime Movie… (Insert an embedded Quicktime movie Player): Invokes the Insert Multimedia dialog with the file filter set to Quicktime file types.
    • HTML5 Movie...(Insert an embedded Quicktime movie Player): Invokes the Insert Multimedia dialog with the file filter set to HTML5 movie types.

  • Slideshow... (Insert a Slideshow): Inserts a slideshow in the XML editor. If a slideshow is already selected, it will open the Edit Slideshow dialog.
  • QR Code Ctrl+Q (Insert a QR Code): Opens the QR Code dialog
  • Rule… (Insert a horizontal ruler): Invokes the Insert Ruler dialog
  • Hyperlink… (Insert a hyperlink): Invokes the Insert Hyperlink dialog
  • Bookmark… (Insert a bookmark): Invokes the Manage Bookmarks dialog
  • Named Destination (Insert a named destination): Invokes the Named Destination dialog
  • Cross-Reference… (Insert a Cross-Reference): Invokes the Insert Cross-Reference dialog
  • Topic Popup… (Insert a Topic Popup): Invokes the Insert Topic Popup dialog
  • Snippet… (Insert a snippet): Invokes the Insert Snippet Link dialog
  • Variable… (Insert a variable): Invokes the Variables dialog
  • Glossary Term Link (Mark selected text as glossary term): Invokes the Create Glossary Term link (must have some text selected for option to be active)
  • Toggler… (Insert a Toggler Control): Invokes the Insert Toggler dialog
  • Drop-Down Text… (Insert drop-down): Invokes the Insert Drop-Down dialog

  • Expanding Text… (Insert an Expanding Text): Invokes the Insert Expanding Text dialog
  • Text Popup… (Insert a Text Popup Control): Invokes the Insert Text Popup dialog
  • Text Box… (Insert a text box): Invokes the Insert Text Box dialog
  • Footnote… (Insert a Footnote): Invokes the Insert Footnote/Endnote dialog
  • Script… (Insert a script): Invokes the Insert Script dialog

  • Help Control: Invokes the Help Control sub-menu

    • Concept Link (A-Link)… (Insert a Concept Link control): Invokes the Insert Concept Link Control dialog
    • Related Topics Control… (Insert a Related Topics control): Invokes the insert Related Topics Control dialog
    • Keyword Link Control (K-Link)… (Insert a Keyword Link control): Invokes the Insert Keyword Link Control dialog
    • Shortcut Control… (Insert a Shortcut control): Invokes the Insert Shortcut Control dialog

  • Proxy: Invokes the Proxy sub-menu

    • Insert Body Proxy… (Insert a Body Proxy): Invokes the Body Proxy dialog
    • Insert Breadcrumbs Proxy… (Insert a Breadcrumbs Proxy): Invokes the Breadcrumbs Proxy dialog
    • Insert Topic Toolbar Proxy… (Insert a Topic Toolbar Proxy): Invokes the Topic Toolbar Proxy dialog
    • Insert TOC Proxy… (Insert a TOC Proxy): Invokes the TOC Proxy dialog
    • Insert Mini-TOC Proxy… (Insert a Mini-TOC Proxy): Invokes the Mini-TOC Proxy dialog
    • Insert Index Proxy… (Insert an Index Proxy): Invokes the Index Proxy dialog
    • Insert Glossary Proxy… (Insert a Glossary Proxy): Invokes the Glossary Proxy dialog
    • Insert Endnotes Proxy… (Insert an Endnotes Proxy): Invokes the Endnotes Proxy dialog

Tools Menu

To access:

  • Click the Tools menu
  • Press [ALT] + T on the keyboard

Tools Menu Items (Tooltips in parentheses)

  • Update Cross-References… (Update Cross-References in the current document): Updates Cross-References in the current document if any require updating
  • Edit Master Stylesheet (Edit the associated master stylesheet document): Opens the Master Stylesheet in the Stylesheet Editor
  • Stylesheet Links (Manage links to stylesheets): Invokes the Stylesheet Links dialog, unless a Master Stylesheet is used by the project or primary target, in which case a message indicating as much will appear
  • Edit Page Layout Document (Edit the associated Page Layout document): Opens the Master Page Layout in the Page Layout Editor

  • Index: Invokes the Index submenu

    • Index Window (Open the Index Window): Opens the Index window
    • Insert "[selection]" as Index Keyword:
    • Index Entry Mode (Toggle Index Entry Mode): Enables or disables Index Entry Mode
    • Auto-complete Index Entries (Auto-complete Index and Concept entries in the Index and Concept windows): Enables or disables auto-complete of index and concept entries in the Index and Concepts panes

  • Concepts: Invokes the Concepts submenu

    • Concept Window (Open the Concept Window): Opens the Concepts window
    • Concept Entry Mode (Toggle Concept Entry Mode): Enables or disables Concept Entry Mode
  • Glossary Terms (Open the Glossary Terms Window): Opens the Glossary Terms pane
  • Page Frame Contents Pane (Open the Page Frame Contents Entry Window): Opens the Page Frame Contents pane
  • Footnotes (Open the Footnotes window): Opens the Footnotes pane
  • Spell Check While Typing (Spell check while typing): Enables or disables spell check while typing
  • Spell Check Window (Open the Spell Check window): Opens the Spell Check window
  • Ignored Words… (Edit the 'ignored words' associated with the current document): Invokes the Ignored Words dialog for the current document
  • Hyphenation Window (Open the Hyphenation Window): Opens the Hyphenation Window
  • Edit Hyphenation Rules… (Edit Custom Hyphenation Rules): Invokes the Open Hyphenation Document dialog

  • Contributions: Invokes the Contributions submenu

    • Contributions (Opens Contributions window): Opens Contributions window
    • Create Contribution Template (Create a Contribution Template using the active topic): Creates a contribution template using the active topic
    • Import Contribution Document (Import a contribution document): Imports a contribution document

  • MadCap: Invokes the Launch submenu

    • Launch MadCap Capture (Launch MadCap Capture): Launches MadCap Capture
    • Launch MadCap Mimic (Launch MadCap Mimic): Launches MadCap Mimic
    • Launch MadCap Analyzer (Launch MadCap Analyzer): Launches MadCap Analyzer
  • Regenerate Captured Images (Regenerate Captured Images): Regenerates all captures images using the props file (in the event the props file was edited outside of MadCap Capture)
  • Manage Mappings: Opens the manage mappings dialog.
  • Synchronize Mapped Files: Opens the synchronize files dialog.
  • Manage Language Skins (Run the Manage Language Skins dialog): Invokes the Language Skin dialog
  • Manage Templates (Open the Template Manager): Invokes the Template Manager dialog
  • Options (Change Global Application Options): Invokes the Options dialog
