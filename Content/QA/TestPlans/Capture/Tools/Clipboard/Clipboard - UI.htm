UI

 

Send To Clipboard: Copies everything (canvas and objects) to the Clipboard. Whereas if you hit Select All > Copy > Paste, only objects are copied.

Paste: Pastes last object/image to the Clipboard

Cut: Cuts and copies object/image to the Clipboard

Copy: Copies object/image to the Clipboard

Delete: Deletes object/image and never sends it to the Clipboard

Auto Capture to Clipboard: Once you Capture something, you should be able to paste right that image into any application
