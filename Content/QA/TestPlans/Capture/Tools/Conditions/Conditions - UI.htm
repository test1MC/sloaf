UI - Creating and Editing Conditions:

File Properties | Condition Tags

  • To edit a field in the name column, simply click it and start typing
    • Alternatively, select the field and press F2 to start editing

  • To edit a background field, select it and pick a color from the Pick Color... dialog.

  • To edit a field in the Comment column, simply click it and start typing
    • Alternatively, select the field and press F2 to start editing
  • To Create a new condition, click in the Name field of the bottom row and give the condition a name. 
    • As soon as you start typing, a unique color for the condition should be provided automatically
  • To delete a condition, select the Name and press delete

 

UI - Using Conditions:

Project

File Properties | Conditional Text

When working in the context of a Project conditions are created, maintained and included/excluded at the Project level instead of the image level.

  • The Tags section will list the available Conditions will be labeled as Project.<Condition Name>, where <condition name> is the name of the condition.

  • The unnamed column shows the associated color of the tag to each condition name
  • The Include column shows which condition tags are set to be "included" in the output
  • The Exclude column shows which condition tags are set to be "excluded" in the output
  • If all options are unchecked, all content will display
  • If one Include item is checked, all other content (flagged by conditions) will not display
  • The top box area is the Include expression area. All conditions set to include will be shown in this box
  • The bottom box area is the exclude expression area. All conditions set to exclude will be shown in this box
  • Clicking on the Include/Exclude column header will select/deselect all options

 

 

Image in a Project

!File Properties | Conditions tab

 

  • Applying a condition to a object will display a colored box on the object.
    • The color will be the color of the condition applied
    • If multiple conditions are applied, the condition indicator will show all colors applied.

 

Stand Alone Image

!File Properties | Conditions tab

When working in a Stand Alone image, the condition tab of the image will function like that of a Project condition tab. Essentially, it becomes the tab where you select what conditions are included/excluded.

  • The Tags section will list the available Conditions will be labeled as Movie.<Condition Name>, where <condition name> is the name of the condition

  • Conditions belonging to a linked Flare project will be labeled as <Flare Tag Set Name>.<Condition Name>, where <Flare Tag Set name> is the name of the Condition Tag Sets linked from Flare and <Condition Name> is the name of the condition
  • The unnamed column shows the associated color of the tag to each condition name
  • The Include column shows which condition tags are set to be "included" in the output
  • The Exclude column shows which condition tags are set to be "excluded" in the output
  • If all options are unchecked, all content will display
  • If one Include item is checked, all other content (flagged by conditions) will not display
  • The top box area is the Include expression area. All conditions set to include will be shown in this box
  • The bottom box area is the exclude expression area. All conditions set to exclude will be shown in this box
  • Clicking on the Include/Exclude column header will select/deselect all options

 

 

Object

Right-Click <object>| Properties | Conditions tab

Applying conditions onto objects control what objects are shown on generated output. Depending what conditions are included/excluded, any number of objects can be included/excluded from the output.

  • Applying a condition to an object will display a colored box on the box on the object in the main editor.
    • The color will be the color of the condition applied
    • If multiple conditions are applied, the condition indicator will show all colors applied
  • The timeline pane will also indicate when an object has a condition
    • The Object will be the color of the condition applied
    • If multiple conditions are applied, the object will have a checkered pattern with the colors used
  • If this object is excluded from output, the movie will generate the movie without this object
    • Any actions associated with the object will not occur

UI - Indicator Toggle:

In the main editor window, clicking the button will toggle on/off the conditional indicators you see on the objects. This will not effect the output in any manner.
