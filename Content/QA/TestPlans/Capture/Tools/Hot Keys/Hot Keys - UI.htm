Ctrl+J (Configure Hot Keys )

UI 

The selected item in the Keyboard Shortcuts dialogue can be assigned keys from the key assignment column on the right and the modifier key assignment under the available commands.

Configure Hot Keys

Options | Hot Keys

 

UI Controls

  • Action: Shows Options and assigned Shortcuts
  • Key Assignment: Select the hot key to assign to the selected item
  • Modifier Key Assignment: Associate Control/Shift/Alt keys to the selected key assignment
    • Shift: Selects shift key
    • Control: Selects control key
    • Alt: Selects alt key
  • Disable Hot Keys: Disable the hot keys
  • Reset to factory defaults: Returns all shortcuts to the default settings.

Reset to defaults - UI

UI Controls

  • Yes/No: Confirm resetting to defaults.
