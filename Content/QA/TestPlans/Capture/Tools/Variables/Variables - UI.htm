Variables tab

UI controls

  • Variables: Displays a list of all Variables

    Columns:

    • Name: Displays the name of all Variables
    • Definition: By default, displays the definition of the variable. Double-clicking the Definition here lets you edit the variable definition specifically for this target.
    • Comment: Displays the comments

Test Cases

How to access variables in Capture:

  • In any Text Edit mode or Rich Text Editor, there will be a standard Variables icon which brings up the Variables dialog.
    • Both accessible in Object Properties
    • Text Edit mode:
      • Also automatically comes up when inserting a new Text Rectangle
      • or hitting F2 on objects

How to link your Capture Image to a Flare Project:

Edit with Capture from image in Flare

  • They automatically links and provides the image with Flare variables

In Capture:

  • Open the movie
  • !File Properties.
  • Select the Project tab.
  • Click the Browse button.
  • In the dialog that opens, select and double-click the Flare project to which you want to link.

Using Shared Variable and Conditions only works one way. They must all be created within Flare to use them in Capture. Variables and Condition Tags created in Capture are not shared with your Flare Project, however they will output correctly.
