Status Bar:

Appearance

  • Background Section: Controls the look of the background
    • (default): Will set the background back to the default state
    • (as is): Will leave the background as it currently is. (Useful for multiple selections)
    • Solid:  Will set the chosen color as the only color for the background
    • Top to Bottom: Will set a top to bottom gradient, based on the two colors chosen
    • Bottom to Top: Will set a bottom to top gradient, based on the two colors chosen
    • Left to Right: Will set a left to right gradient, based on the two colors chosen
    • Right to Left: Will set a right to left gradient, based on the two colors chosen
    • Top Left to Bottom Right: Will set a top left to bottom right gradient, based on the two colors chosen
    • Bottom Left to Top Right: Will set a bottom left to top right gradient, based on the two colors chosen
  • Fill Start: Controls the primary color
    • (default): Is no color
    • (as is):
    • (transparent): Select this if you want one of the gradient colors to be transparent
    • More Colors: Launches the Color Picker Dialog
    • Pick Screen Color: Launches the “screen color picker” and allows the user to pick a color currently viewable anywhere on the screen
    • Color Grid: Displays 132 colors from the Windows color control – Should always display the same available colors.
  • Fill End: Controls a secondary color (used for gradient styling)
  • Border Section: Controls the look of the border
    • Color:  Sets a color for the border
    • Width:  Sets the thickness, in pixels, for the border
  • Padding Section: Controls the padding. (The space around the text)
    • Left: Sets the padding on the left side of the object
    • Right: Sets the padding on the right side of the object
    • Top: Sets the padding on the top side of the object
    • Bottom: Sets the padding on the bottom side of the object
    • Copy Down:  Will copy the value in the Left field down to all the other fields
  • Canvas Size Section:
    • Minimum Width: Sets minimum width to an image if capture is smaller
    • Minimum Height: Sets minimum height to an image if capture is smaller
    • Canvas Auto-Adjust to Include Objects: Toggle off if you want to only manually resize the canvas edges
  • Preview Section: Will display a preview of the object
    • This area will update on the fly as properties are selected

Format

  • Medium: Web is Default.
    • Can select Print or even a custom (Must Enable Format if selected)
  • Format: you can save it using one of the following file formats: BMP, HDP, JPG, JPEG, GIF, PNG, TIF, TIFF, WDP, XPS.
  • JPEG Quality: controls the quality of JPEG if that is the format selected
  • Color Depth: When you capture an image, you can specify the color depth (i.e., how many colors to use when displaying the image). "Highcolor" images use 16-bit color depth. "Truecolor" images use 24-bit or 32-bit color depth.
    • Gray Scale: changes format to gray scale
  • Print DPI:determines how many pixels are used to display an image in a given linear inch.
    • Lock DPI
