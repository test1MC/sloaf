Shape Groups - UI

General
  • Shape group is a group of objects that have been "Stuck" together to essentially form one object
  • Clicking the Group Selected Items toolbar item on the local frame toolbar, while you have multiple object selected, will group the selected objects into a shape group
    • This toolbar item is disabled until multiple objects are selected
    • Object is realizable by the 8 resize points around the object
  • Right-Clicking an object after making multiple selections and selecting Group Selected Items will group the objects into a shape group
  • Right-Clicking a shape group object and selecting Ungroup Shapes, will remove the grouping
    • All objects within the group will still remain, maintaining all settings each object had prior to the grouping
  • Clicking and dragging the shape group will drag the shape group object to a new location
  • All objects will maintain individual properties even while in a shape group, unless the shape group applies the same property
    • For example: A rectangle object with a trajectory, will still follow its own trajectory UNLESS the shape group itself has a trajectory set
  • Holding Shift and grabbing corner handles rotates existing objects or groups at perfect angles based on 15 degrees
    • Does not works initially when drawing (feature request in for that)
