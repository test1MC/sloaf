Basic Appearance

 

General
  •  Clicking the Arrow toolbar item on the frame toolbar will let you draw an arrow object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 2 resize points around the object
  • Clicking and dragging the arrow will drag the whole object, including the pointer
  • Pressing F2 will bring up a rich text window
  • Typing with the object selected will bring up a rich text window
  • Holding Shift draws straight arrows or rather perfect angles based on 15 degrees
    • Works when initially drawing the arrow
    • Does not works when modifying existing arrow
