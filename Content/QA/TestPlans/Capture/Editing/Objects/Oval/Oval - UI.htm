Basic Appearance

General
  • Clicking the Oval toolbar item on the local frame toolbar will let you draw a oval object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
  • Clicking and dragging the oval will drag the oval object to a new location
    • Pressing F2 will bring up a rich text window
    • Typing with the object selected will bring up a rich text window
  • Holding Shift while manually rotating existing objects or groups rotates them at perfect angles based on 15 degrees
    • Does not works initially when drawing (feature request in for that)
