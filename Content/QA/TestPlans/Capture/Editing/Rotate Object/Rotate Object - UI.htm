Rotate Object - UI

Opening the UI

!Object | Rotation group

!Format toolbar | Rotate

Right click over objects | Rotate

Hold down CTRL over a selected objects corner handles to Manually Rotate

UI

Ribbon

 

UI Controls

Rotate Manually: Full control over degree of rotation

Reset Rotation: Reset the rotated object

Rotate Clockwise: Rotate object clockwise by 1 degree

Rotate 45 Clockwise: Rotate object clockwise by 45 degrees

Rotate 90 Clockwise: Rotate object clockwise by 90 degrees

Rotate Counter Clockwise: Rotate object counter clockwise by 1 degree

Rotate 45 Counter Clockwise: Rotate object counter clockwise by 45 degrees

Rotate 90 Counter Clockwise:Rotate object counter clockwise by 90 degrees
