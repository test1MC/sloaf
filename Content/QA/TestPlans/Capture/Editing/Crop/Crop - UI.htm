Selection Rectangle - Quick Crop Method

  • Select an area within the image/canvas with the Selection Rectangle.
  • Then right click to open the context menu
  • Choose Crop Area which accepts the cropping region defined by Selection Rectangle.

 

Manually Resizing the Canvas

UI Controls:

  • Canvas handles: white square knobbies around the canvas edge
  • Knobbies should always remain the same size no matter the zoom level
  • Knobbies can be grabbed and moved in any direction
    • Should not go past the opposite handle on other end (Should not be able to invert the canvas)
  • Adjusting canvas should act like cropping
    • Therefore it should crop according to Discard History is on or off
  • The handles and outline represent the edges of the true canvas
    • Anything outside is just auto-padding

Discard History

Can be toggled on/off:

  • !Edit | Discard History (Tool Strip and Ribbon)
  • Also found in local toolbars for Crop Image and Crop Selection

Discard History On

 

  • With Option toggled on, previous background history is not retained (new behavior in v7).
  • Any areas of the original image should not appear if they were previously cropped out.

 

Discard History Off

  • With the option off (default setting), previous background is retained and can be viewed when readjusting crop region or canvas handles.
  • Full image shows each time you re-crop.
  • As you move the cropping region (orange color) a green history outline remains for reference of your last crop.

Test Cases

  • When cropping a full image, only the true base canvas area can be cropped, excluding any auto extended areas.
  • Make sure whatever state of Discard History is respected when reopening file.
  • Make sure whatever state of Discard History is respected when hitting undo.
  • Try switching between Discard History on and off.
  • Make sure behavior for Crop Image, Crop Selection (Image Objects), and manually adjusting canvas are the same
  • Make sure you can create image objects via Magic Wand or Selection Rectangle and be able to crop them just like imported image objects.
  • Make sure that after cropping that any further changes to canvas or placements of objects respond as expected.
  • Cropping history should be remembered in the undo buffer
    • If you undo, then the previous crop should be remembered next time you crop.
