Medium: Can select between Web (Default) and Print by default. But can also select any custom medium too.

Scale: Alter the scale of the image (e.g., 1 = 100%)

Print DPI: Dots Per Inch

Lock DPI (Print DPI): Lock the DPI

Print Size: Width and Height in print

Enable Preview: Turn preview on and off
