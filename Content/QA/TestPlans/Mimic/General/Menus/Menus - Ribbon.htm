Menus - Ribbon

Accessing the Ribbon (If Ribbon is not enabled)

  1. Go to !Tools | Options
  2. In the "Interface" tab, select "Ribbon"
  3. Click OK

 

Menus are located at the top of the application, above all the toolbars

File Menu

To access:

  • Click File
  • ALT + F

File Menu Items (Tooltips in parentheses):

 

  • New (Create a new file):Invokes the New dialog.
  • Open (Open file):Invokes an Open dialog which lets you browse for appropriate file types to open in the application
  • Save (Save file):Invokes Save options.
  • Record Movie: Invokes the recording frame
  • Movie Collection: Invokes Start New Collection Wizard
  • Movie From Media File: Invokes File Explorer
  • Movie From Screenshot: Invokes the Invokes the recording frame
  • Import PowerPoint: Invokes Import PowerPoint Wizard
  • Blank Movie Invokes Add New Movie dialog.
  • Save (Save the active file): Saves the current file
  • Save As (Save a copy of the active file to a different name, location, and/or file type): Invokes the Save As dialog
  • Save All (Save all files): Saves all files

 

  • Close (Close the active file or selection):Closes the current file
  • Close Collection (Close the active Collection):Closes the Collection
  • Print (Print the active file or selection):Prints the current file

Home Menu

To access:

  • Click Home
  • ALT + H

 

Mode:

  • Hand Mode: Drag areas of a frame around.
  • Select Mode: Select objects in the Frame Editor.
  • Preview Mode: Allows you to use the Frame Editor to view an action set on an object.

 

Clipboard:

  • Cut: Will cut the active object/frame
  • Copy: Will copy the active object/frame.
  • Paste: Will paste whatever is currently on the clipboard.
  • Delete: Deletes the active selection.
  • Select All: Will select all objects on the frame.

Insert:

  • Objects:
    • Rectangle: Inserts a rectangle into a movie frame.
    • Polygon: Inserts a polygon into a movie frame.
    • Oval: Inserts an oval into a movie frame.
    • Polyline: Inserts a polyline into a movie frame.
    • Graphics:
      • Bubble: Inserts a Bubble into the frame.
      • Loop: Inserts a Loop into the frame.
      • Arrow: Inserts an Arrow into the frame.
      • Cursor: Inserts a Cursor into the frame.
    • Interactive:
      • Button: Inserts a Button into the frame.
      • Typing Box: Inserts a Typing Box into the frame.
      • Input Box: Inserts an Input Box into the frame.
    • Text Rectangle: Inserts a Text Rectangle into the frame.
  • Effects:
    • Clip: Applies a clip effect to an object.
    • Opacity: Applies an Opacity effect to an object.
    • Gray Scale: Applies a Gray Scale effect to an object.
    • Blur: Applies a Blur effect to an object.
  • Insert Screenshot:
    • Captured Region: Capture a rectangular region on your screen and insert it into the current frame as an image object.
    • Captured UI Element: Capture a fixed area of a window and insert it into the current frame as an image object.
  • Insert Image: Opens file explorer.
  • Insert Sound: Opens Add Sound to Frame dialog.

 

Appearance:

  • Shadow: Applies shadow to object.
  • Background Color: Opens color picker.
    • Applies background color to selected object.
  • Line Color: Opens color picker.
    • Applies line color to a line applied to a frame.

 

Text:

  • Font: Change font family.
  • Font Size: Change font size.
  • Bold: Bolds selected objects text.
  • Italic: Italicizes a selected objects text.
  • Underline: Underlines a selected objects text.
  • Unformat: Remove formatting for selected objects text.
  • Color: Lets you select a color for the selected content.
  • Left: Will align the text of the selected object(s) to the left
  • Center: Will align the text of the selected object(s) to the center
  • Right: Will align the text of the selected object(s) to the right
  • Top: Will align the text of the selected object(s) to the top
  • Middle: Will align the text of the selected object(s) to the middle
  • Bottom: Will align the text of the selected object(s) to the bottom.

 

Order:

  • Sink:
    • Sink: Will sink the object one layer lower, in relation to the other objects
    • Sink To Bottom: Will sink the object to the bottom layer. All other objects will be above this object.
  • Float:
    • Float: Will float the object one layer higher, in relation to the other objects.
    • Float To Top: Will float the object to the top most layer. All other objects will be below this object.

 

Rotation:

  • Rotate 45 Clockwise:
    • Rotate 90 Clockwise: Will rotate the object 90 degress clockwise.
    • Rotate 45 Clockwise: Will rotate the object 45 degress clockwise.
    • Rotate Clockwise: Will rotate the object clockwise.
  • Rotate 45 Counter Clockwise:
    • Rotate 90 Counter Clockwise: Will rotate the object 90 degress counter clockwise.
    • Rotate 45 Counter Clockwise: Will rotate the object 45 degress counter clockwise.
    • Rotate Counter Clockwise: Will rotate the object counter clockwise.
  • Reset Rotation: will put the object back to its default location (zero degrees rotated).

 

Alignment:

  • Align Left: Will align the selected object(s) to the left.
  • Align Right: Will align the selected object(s) to the right.
  • Align Top: Will align the selected object(s) to the top.
  • Align Bottom: Will align the selected object(s) to the bottom.
  • Align Center: Will align the selected object(s) to the Center.
  • Align Middle: Will align the selected object(s) to the middle.
  • Same Width: Will set the selected objects to the same width (based on the first item selected).
  • Same Height: Will set the selected objects to the same height(based on the first item selected).
  • Same Size: Will set the selected objects to the same size (based on the first item selected).
  • Center on Canvas: Will set the objects to the center of the canvas.
  • Horizontally Center on Canvas: Will set the objects horizontally center on the canvas.
  • Vertically Center on Canvas: Will set the objects vertically center on the canvas.

Advanced:

  • Use Default Look: Will set the object to use the set default attributes.
  • Set Default Look: Will set the currently selected objects attributes as the default for that shape.
  • Apply Style to Similar:
    • Apply Style to Similar: Selected objects attributes are applied to all similar objects on the screen.
    • Apply Style to All: Selected objects attributes are applied to all objects on the screen.
  • Pick Style: Lets you pick a style of the selected shape. Any style you have added to your palette, or a factory style can be chosen.
  • Delete Point: Delete points for polylines and polygons.

Frame Menu

To access

  • Click Frame
  • Alt + R

 

 

Navigate:

  • Previous:
    • Previous Frame: Go to the previous frame.
    • Next Frame: Go to the next frame.
  • Next:
    • Next Frame: Go to the next frame.
    • Last Frame: Go to the last frame.

 

Edit:

  • Copy: Will copy the selected frame to the clipboard.
  • Paste: Will paste the frame currently located on the clipboard.
  • Remove: Will remove the selected frame.
  • Copy Background: Will copy the current frames background to the clipboard.
  • Paste Background: Will paste the current background on the clipboard to the selected frame.

 

Insert:

  • Blank: Will insert a blank frame.
  • Duplicate: Will make a duplicate of the selected Frame.
  • PowerPoint: Will bring up the insert PowerPoint dialog.
  • Screenshot: Will trigger recording/capture frame.
  • Recording: Will trigger recording/capture frame.
  • Insert Video Frame: Will trigger file explorer to browse for video files.

 

Reuse:

  • Add to Master Frames: Will add the currently selected frame to the master frames list.
  • Add to Library: Will add the current frame to the active Frame Library.

 

Properties:

  • Properties: Will open up the frame properties dialog.

Tools Menu

To access:

  • Click Tools
  • Alt + T

 

 

Zoom:

  • Zoom In: Increases the scale of the image each time you select the option.
  • Zoom Out: Reduces the scale of the image each time you select the option.
  • Scale to 100%: Resets the scale of the image to 100%.
  • Fit Width: Resizes the frame to fit the width of the current size of the Frame Editor.
  • Fit Height: Resizes the frame to fit the height of the current size of the Frame Editor.

 

Object:

  • Conditional Indicators: Enables conditional indicators.
  • Preview Conditions: Enables Preview Conditions.
  • Anchors: Shows anchors (arrows) on each side of an object.

 

Grid:

  • Snap to Grid: Enables Snap to Grid.
  • Show Grid: Enables grid in frame editor.
  • Edit Grid: Triggers Grid Options dialog.

 

Language:

  • Check Spelling: Will open/focus on the Spell Check pane.
  • Edit Dictionary: Will open/focus on the Open Dictionary pane.
  • Ignored Words: Will open/focus on the Ignored words dialog.

 

Window Menu

To access:

  • Click Window
  • Alt + W

 

 

Open Windows:

  • Window:
    • (dropdown shows current mimov tabs open in movie editor)
    • Windows...: Triggers Manage Windows dialog.

 

Active Pane:

  • Dock: the current window becomes "docked" (or "attached").
  • Auto Hide: "Pins" the current pane to the side (or bottom or top) of the interface so that you have more room for your workspace.
  • Float: he current window is no longer "docked" (or "attached") to the application; instead, it becomes a free-floating element in the interface.

 

Window Layout:

  • Layout:
    • (shows any saved Windows layouts)
    • Layouts...: Opens Manage Window Layouts.
  • Save Layout:
    • Save Window Layout: save the changes to the layout.
    • Save Window Layout As: Opens the Rename Window Layout dialog.
  • Reload Layout: select this option to return to the saved configuration of the layout.
  • Reset Layout: Returns the current configuration of the interface to the factory layout settings.
  • Auto-save Layout: Automatically saves the layout of the workspace.

 

Help Menu

To access:

  • Click Help
  • Alt + E

 

 

Topics:

  • Dynamic Help: Opens the Dynamic Help pane.
  • Help Topic: Opens a Help Topic.
    • New Help Topic Window: Opens a second Help Topic (only active if one Help Topic is already open).

 

Contents:

  • Contents: Opens the Help TOC pane.
  • Locate in TOC: Opens the Help TOC (if it is closed) and highlights the current Help topic.
  • Auto-sync TOC: Enables auto-sync in the Help TOC.

 

Panes:

  • Index:
    • Index: Opens the Help Index pane.
    • Index Results: Opens the Index Results pane.
  • Search: Opens the Help Search pane.
  • Favorites: Opens the Help Favorites pane.
  • Glossary: Opens the Glossary pane.

 

Tutorials:

  • PDF Guides: Invokes the Guides submenu which displays all PDF guides available for the application.
  • Videos: Opens the Help Topic "Video Tutorials and Movies" from the internal Help.

 

Community:

  • Help Community: Launches the default web browse and opens the MadCap Software Forums page.

 

Product:

  • Renew Maintenance: Opens Madcapsoftware page for renewing maintenance.
  • Activate Product: Invokes the Activate Product Wizard.
  • Deactivate Product: Deactivates this product on your machine.
  • Register Product: Invokes the Register Program Wizard.
  • Check for Updates: Invokes the New Updates Available dialog (or the No Updates Available dialog).
  • About: Invokes the About box.
  • Report a Bug: Opens Madcapsoftware page for bug submissions.
