Debug Ribbon

The debug ribbon can be used during development for testing features that are not yet visible in the daily builds.

Enable Debug Ribbon

  1. Navigate to \\fs1\QA
  2. Double-click EnableDebugMode.reg
  3. Relaunch Flare and you will see Debug in the Ribbon

 

Disable Debug Ribbon

  1. Before you disable the ribbon, if you had checked any additional boxes in this ribbon, uncheck them so they are back to default.
  2. Open your registry
    1. Start
    2. type "regedit" > Enter > Yes to User Account Control prompt
  3. Navigate to HKEY_CURRENT_USER\Software\MadCap Software\Shared
  4. Right-click the DebugMode registry entry > Delete
