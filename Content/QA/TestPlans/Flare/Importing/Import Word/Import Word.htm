Import Word

Bug Base Product Area:

  • Import: Word

 

Description:

Gives the users the ability to import and reimport Word files (.doc .rtf and .docx) and control the conversion of styles, formatting, and links during the conversion to XML. You can create a new project or add topics to an existing project by importing Word files. Special case: Word import to Word output “round tripping” will not produce an exact file match. Import is designed to create xml files that can be used in all outputs. Word specific formatting might be removed or might not map to a specific element in Flare.
