Import DITA

Bug Base Product Area:

  • Import: Dita

 

Description:

You can create a new project or add topics to an existing project by importing DITA files. Flare lets you import DITA topic files (*.dita) or DITAMAP files (*.ditamap). DITA topics are converted to XML topics in Flare and DITAMAP files are converted to Flare TOC files. Multiple DITAMAP files can be imported into a single Flare project however, if you want to import two DITAMAP files that exist in unique directories, you need to create a unique DITA Import file for each (multiple DITAMAP files that exist in the same directory can be imported together in the same DITA Import file).
