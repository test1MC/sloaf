Import Frame

Bug Base Product Area:

  • Import: Frame

 

Description:

Gives the users the ability to import and reimport Adobe FrameMaker files (MIF, FM and Books) and control the conversion of styles, formatting, and links during the conversion to XML. You can create a new project or add topics to an existing project by importing FrameMaker files. Special case: Frame import to Frame output “round tripping” will not produce an exact file match. Import is designed to create xml files that can be used in all outputs. Frame specific formatting might be removed or might not map to a specific element in Flare.
