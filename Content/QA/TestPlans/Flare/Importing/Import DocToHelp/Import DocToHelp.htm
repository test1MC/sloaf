Import DocToHelp

Bug Base Product Area:

  • Import: DocToHelp

 

Description:

Flare can create a new project based on an imported Doc To Help project (*.d2h). Flare will only convert HTML5 source documents, so user will need to use the D2H converter to convert all Word source documents. This included any Rich Text Variables (Project Ribbon | Variables | Rich Content Variables) authored in Word. If the D2H source document is a multiple topic type, upon import it will be split into Flare topics by heading levels h1 – h5. All documents converted from Word to HTML5 are multiple topic types. The splitting is used to preserve the look of the online output with each heading level showing as a new page in the online TOC. To avoid this behavior, the user can convert their multi topics into single topics using D2H UI.

 

For additional notes, See The Imported Project
