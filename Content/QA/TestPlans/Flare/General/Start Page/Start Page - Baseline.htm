Start Page - Baseline

Verify the following:

  • Start Page can be closed with !File | Close (broke in v10: bug 100392)
  • !Window | Close All Documents options should not close the Start Page
  • The Start Page should not display in the Manage Windows dialog (!Window | Windows...)
  • The Start Page should not appear in the Recent Files list (!File | Recent Files)
  • When the Start Page is in focus, !File | Review | Send for Review should be inactive
  • When the Start Page is in focus, !File | Save | Save As Contribution Template should be inactive
