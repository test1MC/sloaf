Language Skins - Editor (Editing Language Skins)

Add a Language Skin File

 

Language Skin files are added to the project in the Advanced folder of the Project Organizer:

  • Project Organizer > right-click Advanced folder > Add Language Skin.

 

Editing a Language Skin File

Language Skin files are opened/editied by double-clicking the Language Skin file in the Advanced folder.

Language Skins Editor - UI Test Cases

UI Controls

  • Skin Language: Dropdown menu displaying available skin languages in Flare.

  • Language Skin Targets: Select the check boxes for the target types you want to modify. Translatable values appear in the grid.
