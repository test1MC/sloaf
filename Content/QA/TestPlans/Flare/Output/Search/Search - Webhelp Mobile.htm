Search UI and Options- WebHelp Mobile Help

Enabling Search and how it works in the output

  1. Search is always enabled in the output. Hiding it does not remove search features in the output.

  2. Associate the skin with search enabled with the target that is going to be generated.

    • Options in HTML Target that relate to search

      Performance Tab

  3. How search looks in the output

    • Topic Titleis shown and results are ranked but number is not shown.

    • When topic is opened from the search results, searched term is highlighted

Search Filter Sets are not supported for WebHelp Mobile output

WebHelp Mobile has two types of outputs after build - the basic and advanced versions of the output. The basic version of the output does not support search but the advanced version supports search
