Search UI and Options- HTML5 Skinless output

Enabling Search

  1. Search is always enabled in the output.  Hiding it does not remove search features in the output.

    Add a search bar proxy to the project to provide UI for searching the output. This is best added to a masterpage so that it is always accessible to the user.
