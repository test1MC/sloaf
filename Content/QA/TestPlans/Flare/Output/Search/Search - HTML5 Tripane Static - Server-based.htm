Search UI and Options- HTML5 Tripane Static output

Enabling Search

  1. Search is always enabled in the output. Hiding it does not remove search features in the output.

  2. Associate the skin with search enabled with the target that is going to be generated.

    • Options in HTML5 Target that relate to search

      Advanced Tab

      Performance Tab

  3. Searching the server-based output requires extra steps - see steps for this at See Output WebHelp Plus - Setup

  4. How search looks in the output

    • Without a search filter

    • With a search filter

    • With a search filter selected - note the filter name is shown

    • Pagination appears at the bottom of the page when the total number of results exceeds the page per results count in the target

    • Searched term is bolded in the search results

    • Searched terms are highlighted when the topic is open from a search result
