Previewing Topics

Dropdown in the Local Topic Toolbar that allows you to preview your topic based on the output format used in any of the targets in your project. By default, it takes any conditions applied to your target and applies them in the preview.

How to Preview Topics

  1. Open the topic that you want to preview.

  2. Do one of the following, depending on the part of the user interface you are using:

    • Local Toolbar In the local toolbar of the XML Editor click . If you click the face of the button, the topic preview is shown based on the format specified in the primary target. If you click the down arrow, you can select any of the targets in your project from a menu. The topic preview is then displayed using the output format specified in that target.

    • Keyboard Shortcut To preview using the primary target, press CTRL+W

 

 

Conditional Text dialog

When you preview for a specific target, the condition tags set for that target are automatically used in the preview. However, if you want to test the output with certain condition tags included or excluded, click the 'Modify Conditional Expression' button in the upper-right corner of the dialog.

 

 

 

Next to each condition tag, select either the Include or Exclude check box, depending on whether you want the content with the corresponding tag to be seen in the output.

As you make your selections, the area at the bottom of the window indicates which tags will be seen in the output.

Note: The topic preview feature is not available for Microsoft Word or Adobe FrameMaker formats.
