List-Of Proxy - Output

Support in Web Outputs:

Support in HTML5 Outputs:

 

  • The generated List of proxy will list the content of elements using the specified style class
    • Test using different root styles/style classes: Headings, paragraphs, th/td, title, etc.
  • For Web output, the content using the specified class will be displayed in the order that it is used.
    • If a topic contains multiple instances of p.test, the list of proxy will display the content of each instance in the order it appears.

Support in DITA Output:

  • DITA does offer an equivalent element to a List-Of Proxy. When generating DITA output, confirm that we simply strip the proxy from the topic. Verify that the output topic which contained the stripped List-Of Proxy is output as valid DITA.

Compiler errors:

  • Including a List-Of Proxy in DITA output:
    MadCap:ListOfProxy found in file: "Topic.htm". It will be removed from the output.
