Stylesheets - Editor

Opening a Stylesheet File

Any Stylesheet file:

The Master Stylesheet file associated with a particular Target:

  • Target Editor: In the General tab of the Target Editor, select a Master Stylesheet and click the "Edit" button.

The Stylesheet file associated with a particular Topic:

  • Right-Click a topic > Properties. In the Topic Properties tab, select a Topic Style Class and click the "OK" button.

Stylesheet Editor (Local Toolbar)- UI Test Cases

 

 

Button (Tooltips in Parentheses):

  • (Switch to advanced view) Button to toggle between Simplified and Advanced views.
  • (Add a new style Selector) Open "New Selector" Dialog.
  • (Run the font set manager) Opens "Font Set Manager" dialog.

    • (Import Styles from other stylesheets) Import styles from other stylesheets.
    • (Rename (F2) ) Rename the selected class
    • (Add a new Medium to the stylesheet) Add a new medium to the stylesheet
    • (Delete medium from the stylesheet) Deletes selected medium from the stylesheet
    • (Manage links to stylesheets) Opens "Stylesheet Links" dialog
    • (Disable Styles) Opens "Disable Styles" dialog

Add Selector

New Selector dialog- Default

 

UI Controls

  • HTML Element Specify the new medium name
  • Class Name Add new css class to selected selector
  • Advanced Options Button to expand advanced options
  • OK Accepts the new medium entry
  • Cancel Closes the dialogue/does not add a new selector

 

New Selector dialog- Advanced Options

 

UI Controls

  • Advanced Selector Displays selected selector
  • Pseudo Class Add Pseudo class to the selected selector
  • Pseudo Class Expression Add Pseudo class function-like syntax ("nth-" and "not" selectors only)
  • Pseudo Element Add Pseudo element to the selected selector
  • Identifier (ID) Add new css Identifier (ID) to selected selector
  • Comments Add comment to new selector
  • OK Accepts the new medium entry
  • Cancel Closes the dialogue/does not add a new selector
