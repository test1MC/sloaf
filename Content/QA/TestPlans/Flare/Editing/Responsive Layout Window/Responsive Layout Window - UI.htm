Responsive Layout Window - UI

 

 

 

Responsive Layout Window Local Toolbar

Toolbar Items (Tooltips in parentheses)

  • Web (Switch to Web Layout Mode): Changes the XML editor to Web Layout mode.
  • Tablet (Switch to Tablet Layout Mode): Changes the XML editor to Tablet Layout mode.
  • Mobile (Switch to Mobile Layout Mode): Changes the XML editor to Mobile Layout mode.

 

Responsive Layout Window UI Controls (Tooltips in parentheses)

 

Row

  • New Style (New Row Style): Open the New Row Style dialog (see below) which lets you create a new style based on a factory template or previously created style.
  • Insert Row (Insert Row): Insert a new row into the open content file.
  • Delete Style (Delete Style): Deletes the style selected in the styles field.

 

Manage

  • Add Cell (): Adds a new cell to the right edge of the row.
  • Delete Cell (): Deleted the selected cell from the layout.
  • Cell Width (): Select a number of columns to set the width of the current cell (in percentage).
  • Cell Offset (): Select a number of columns to provide the offset (left margin) for the current cell (in percentage).
  • Row Gutter (): Enter the number of pixels for a gutter in the entire row.
  • : Moves the entire row up.
  • : Moves the entire row down.
  • : Moves the active cell to the left in the row.
  • : Moves the active cell to the right of the row.

 

 

Cell Style

  • Edit (Edit the CSS style): Opens the stylesheet editor for the selected style for editing.

 

Settings

  • Auto-save Stylesheet: Any changes made to the selected style in this window pane will be applied in the stylesheet. When disabling the option you must save changes in both the XML Editor and Stylesheet Editor.
