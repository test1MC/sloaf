Absolute Positioning-Editing

Text Wrap Context Menu

 

Using the context menu will set local styles to "float:left;" for online media and "Madcap:Mediastyle="@media print {position: absolute; z-index:0;}", with the "z-index" property value (0, 1, -1 Square, in front, or behind)

 

Text Wrap Properties

  • In Line with Text: Resets absolute position options back to the default inline setting.
  • Square:
    • Layout(Print): Wraps text around the absolutely positioned element on any or all sides ( MadCap:mediastyle="@media print { position: absolute; z-index: 0; }")
    • Layout(Web): Sets the Element to "float:left;" (float: left;)
  • Behind Text: Places the absolutely positioned element behind text. (position: absolute; z-index: -1;)
  • In Front of Text: Places the absolutely positioned element in front of text. (position: absolute; z-index: 1;)

 

StyleSheet

This also can be set in the "Positioning group" in a stylesheet

Currently, Setting an element to Text Wrap:Square in a stylesheet will display that element as Text Wrap: In Front of Text in web layout

Elements to test

  • Images
  • Multimedia
  • QR Codes
  • Divs
  • Paragraphs

Currently, Absolute Positioning is not supported for SlideShow, Snippets, and Tables

Note: Paste will place the element at it's defined "top" and "left" position

Test Cases

  • Ensure all Absolutely Positioned elements display correctly when spanned across multiple columns in a multipage layout.
  • Ensure all Absolutely Positioned elements display correctly in Headers, Footers, and Decoration Frames in a page layout.
  • Ensure Absolutely Positioned Expanding Text and Text popups are converted to footnotes in all print outputs.
  • Ensure all Absolutely Positioned text-based outputs can be resized (Knobbie Controls).
  • Ensure you can select an element that has been set to "Behind Text" by placing cursor over element and clicking five times.
  • Ensure Find/Replace works correctly on text that has been Absolutely Positioned.
  • Ensure Spell Check works correctly on text that has been Absolutely Positioned.
