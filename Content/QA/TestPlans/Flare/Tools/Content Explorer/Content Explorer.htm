Content Explorer

Bug Base Product Area:

  • Content/Project Explorer

 

Description:

The Content Explorer is used to hold all of the content-related items used in the project (topics, images, stylesheets, table stylesheets, master pages, snippets, and MadCap Echo files*). The panel is also dockable.

* The Content Explorer may also house linked CHM files for imported RoboHelp projects. These will be covered in the RoboHelp Import and HTML Help (CHM) Output test plans.
