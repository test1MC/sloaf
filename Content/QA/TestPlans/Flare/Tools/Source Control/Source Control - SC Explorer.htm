Source Control - Explorer

The Source Control Explorer is used to view and manage source control-related files and settings for your project. The Source Control Explorer collects all source control information in one place so you do not need to open multiple windows or tabs when working with projects that are bound to source control. It does not replace the Source Control ribbon or Pending Changes window; instead, it gathers all of this information in one place. Here, you can bind and unbind a project, see pending changes, import a project from source control, and manage branches (when using Git).

Settings Pane

  • Provider Type:View the source control provider you are using for your project.

  • Database:View the source control database where your project is bound.

  • Database Folder:View the source control database folder where your project is bound.

  • Unbind:Unbind your project from source control. This removes project information from the Settings fields.

  • Bind:Bind your project to source control. This fills in project information in the Settings fields.

 

 

 

 

 

 

 

 

 

 

 

Pending Changes Pane

  • Commit/Submit/Check In:Send your local files to source control. The terminology on this button changes depending on the source control provider you are using.

  • Revert/Undo Check Out:Return your local files to their most recent state. The terminology on this button changes depending on the source control provider you are using.

  • Comment:Add a comment to the commit/submit/check in. You can view the comment in the file's history.

  • Included Changes:The files in this section will be included in the next commit/submit/check in. You can right-click any file or folder to add it to the Excluded Changes section.

  • Excluded Changes:The files in this section will be excluded from the next commit/submit/check in. You can right-click any file or folder to add it to the Included Changes section.

 

 

 

 

 

 

 

 

 

Branches Pane

  • Published Branches:Shows available published branches.

  • Unpublished Branches:Shows available unpublished branches.

  • Create Branch:Create a new unpublished branch.

  • Delete Branch:Delete the selected branch. Deleting the branch also deletes any commits on that branch.

  • Select Branch:Switch to the selected branch.

  • Publish Branch:Publish the selected branch.

Right-Click Menu

Content file right-click menu:

 

  • Menu items are correct
  • Spelling and Grammar are correct
  • No duplicate accelerators
  • All accelerators function
  • Keyboard can navigate and select options
  • Menu is easy to use and understand

 

Right-click menu items:

  • Open: Opens the selected Content file.
  • Commit: Check's in the selected file/folder into Source Control.
  • View History: You can view the history (change sets) of a particular source control file.
  • Show Differences: Shows the differences between your local copy and the newest Source Control copy.
  • Get Latest Version: Get's the latest version of the selected file/folder from Source Control.
  • Exclude: Excludes the selected file from the commit.
  • Exclude Unselected: Excludes all unselected files from the commit.
  • Revert: Returns the file to its previously committed state and does not commit any of your new changes to source control.

Keyboard Navigation

  • Up Arrow and Down Arrow move the selection to the next displayed node.
  • Page Up and Page Down will move the selection to the very first node or the very last displayed node.
  • Home or End will move the selection to the very first node or the very last displayed node.
  • Insert does nothing.
  • Delete does nothing.
