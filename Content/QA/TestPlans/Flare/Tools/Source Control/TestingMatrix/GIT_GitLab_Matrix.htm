Git - GitLab Matrix

Authentication Testing

Name and Email are required for binding to a local repository database - They are not authenticated. They do identify your user when collaborating with other on a Git Remote site.

Log in for binding to a Remote Repository does authenticate credentials specific to the Cloud-based Git Remote Repository site that you are pushing to.

Functionality Testing

Before proceeding with the following tests, install Tortoise Git (https://code.google.com/p/tortoisegit/) so that status of files in the repository can be checked with Git Repo Browser. Installing Git.exe (http://www.git-scm.com/) is also recommended. With that install, you can access local repo using CMD.exe, GitBash, or GitGUI.
