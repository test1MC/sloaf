Build Window - UI

Opening the Build Window

The Builds window pane opens automatically when you generate a target.

It can be opened manually by the following

  • Press F6 to build the primary target which will invoke the build window (or shift the focus if it is already open)
  • Ribbon interface:

    • !View>Builds or Project>Builds.

  • Toolstrip interface: !Build>Builds.

 

 

Build Window UI Test Cases

The build window will open by default as the bottom pane (Under the XML editor)

 

 

Targets selected to build will display in the build window and will remain in view until 'Clear Finished' is selected.

Local Toolbar Buttons (Tooltips in parentheses)

  • Clear Finished: Clears finished builds from the list
  • View Output: View the selected target
  • Open Build Log: Open Build log file
  • Stop Build: Stop the selected build
  • Rebuild Target: Rebuild the target
  • Build Targets: Select targets from a project file to build
  • Open Output Folder: Open the output folder of the selected target in Windows Explorer
  • Clean Target: Delete all the generated output files for the selected target

Build Targets UI

Selecting Build Targets will invoke the Select Targets to Build dialogue

UI Controls

  • Target Displays available targets from the selected Flare project
  • Build Checking this option will build the selected target(s)
  • Publish Checking this option will publish the selected target(s)
  • OK Close dialogue/starts builds of selected targets
    • If no targets are selected, the above error message will display

     

     

     

     

     

     

     

  • Cancel Close dialogue without any additional actions
