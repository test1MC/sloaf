Font Families - UI

Font Sets

Font Set Manager Dialog

UI Controls:

  • Add: Opens the Define Font Set dialog.
  • Edit: Opens the Define Font Set dialog with the selected Font Set name displayed in the grayed out Name field.
  • Remove:Removes the selected Font Set.
  • OK: Applies changes and exits dialog.
  • Cancel: Discards changes and exits dialog.

Test Cases

  • Scroll bar should appear if you create many font sets.
  • Double clicking a font set will open the Define Font Set dialog with the selected Font Set name displayed in the grayed out Name field.

Define Font Set Dialog

UI Controls:

  • Name: Enter a name for the font family. If you are editing the font set, this name cannot be changed.
  • (Font Family field): This is the field above the list of fonts on the left side of the dialog. If you know the name of the font family you want to add to your font set, you can enter it in this field.
  • (Available Font Families List): This is the list of fonts on the left side of the dialog. You can add these fonts to the font set.
  • (Select Font Families List): This is the list of fonts on the right side of the dialog. These fonts are part of the font set.
  • [<<<]: Removes the selected font family from the font set.
  • [>>>]: Adds the selected font family to the font set.
  • (Move Up):Moves the selected font up in the order of fonts in the font set.
  • (Move Down):Moves the selected font down in the order of fonts in the font set.
  • OK: Applies changes and exits dialog.
  • Cancel: Discards changes and exits dialog.

Test Cases

  • Click in the Available Font Families list (left) or Select Font Families list (right) and type a letter, it should scroll to a font that begins with that letter.
