Export Project - UI

To Launch the Exporting Project Wizard

[!File | Export Project] invokes the the Export Project Wizard for the tool strip.

[!Project | Save group | Export Project] invokes the Export Project Wizard for the Ribbon.

UI Controls

 

  • Select Content/Project Files Allows all/some files to be selected to be exported.

    • (Checkbox): Displays next to all files in the project to allow selection
  • Back: Returns to the previous dialog in the Export Project Wizard.
  • Next: Proceeds to the next dialog in the Export Project Wizard.
  • Finish: Closes the Export Project Wizard, invokes the Export Progress dialog
  • Cancel: Closes the Export Project Wizard and does not import anything.
