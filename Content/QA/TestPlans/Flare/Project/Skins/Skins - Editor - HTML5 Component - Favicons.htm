Skin Editor (HTML5 Component Favicons): Favicon

UI Controls

  • Master Favicon:
    • Dropdown: Select from
    • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif)
    • Quick help icon:Clicking the Help Book icon will open the Adding Favicons topic in the help.
  • Browser: Preview will display the selected image, or it will be blank if none is selected. Click Option to expand
    • Icon: If you have a Master Favicon, this will default to say (generated) with Size: 16px.  If you manually select an image, it will display the image name and size of that image.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif;*.ico)
    • Icon: If you have a Master Favicon, this will default to say (generated) with Size: 32px.  If you manually select an image, it will display the image name and size of that image.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif;*.ico)
    • Icon: If you have a Master Favicon, this will default to say (generated) with Size: 96px.  If you manually select an image, it will display the image name and size of that image.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif;*.ico)
    • Shortcut Icon: If you have a Master Favicon, this will default to say (generated).  It will not indicate a size. If you manually select an image, it will display the image name.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif;*.ico)
  • iOS/Android: Preview will display the selected image, or it will be blank if none is selected. Click Option to expand
    • App Icon: If you have a Master Favicon, this will default to say (generated) with Size: 180px.  If you manually select an image, it will display the image name and size of that image.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif;*.ico)
      • Quick help icon:
  • Windows: Preview will display the selected image, or it will be blank if none is selected. Click Option to expand
    • Small: If you have a Master Favicon, this will default to say (generated).  If you manually select an image, it will display the image name.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif)
    • Medium: If you have a Master Favicon, this will default to say (generated).  If you manually select an image, it will display the image name.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif)
    • Wide: If you have a Master Favicon, this will probably not be auto generated (unless you had picked a wide Master Image, so it will default to say (none) and show a warning: Please select a rectangular image (recommend 558 x 270 px).   If you manually select an image, it will display the image name.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif)
    • Large: If you have a Master Favicon, this will default to say (generated).  If you manually select an image, it will display the image name.
      • Select Image: Opens Insert Image dialog. From this dialog you can browse for Supported Favicon Files (*.png;*.jpg;*.gif)
    • Web URL: Type the path where the output will be published.
      • Select Variable: You can use a variable in part of your URL path.
      • Quick help icon:
    • Tile Color: Select the background tile color.  If your image has transparency, this color will show in the transparent area.
