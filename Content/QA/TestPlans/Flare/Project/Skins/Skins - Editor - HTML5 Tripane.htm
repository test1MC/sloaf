Skin Editor (HTML5 Tripane) - UI Test Cases

Skin Editor (HTML5 Tripane): Local Toolbar

  • Preview: Click the button to preview an example of what the skin will look like in the generated HTML5 Tripane output
  • Web Medium: Disabled unless in Styles tab. Click the button to edit the styles in the web medium - Live Preview window switches to web medium.
  • Tablet Medium: Disabled unless in Styles tab. Click the button to edit the styles in the tablet medium - Live Preview window switches to tablet medium.
  • Mobile Medium: Disabled unless in Styles tab. Click the button to edit the styles in the mobile medium - Live Preview window switches to mobile medium.
  • Highlight:Disabled unless in Styles tab. Click the button to enable highlighting function Live Preview pane in the Styles tab.
  • Add Class: Disabled unless the TOC Entry or Toolbar Button styles are selected in the Styles list in the Styles tab- Click to add a new class to the selected style in the Styles list
  • Delete Class: Disabled unless a custom TOC Entry or Toolbar Button style class is selected in the Styles list in the Styles tab - Click to delete the style class created in the Style list

Skin Editor (HTML5 Tripane): General Tab

UI Controls

  • Caption: Insert a caption for your skin.
  • Features: Select which features you want to appear in the output. Options include: TOC, Index, Glossary, Browse Sequences.
  • Make Default: Click this button if you want to make the feature currently selected (and highlighted) to be the default pane displayed on the left in the browser output
  • Options:
    • Automatically Synchronize TOC: Select this option is you want to automatically synchronize the TOC.
  • Unreferenced Images: If an image is added to the skin and then later replaced with another image, Flare does not automatically remove the initial image from the Skin file.
    • Keep: (The default behavior) Image that cease to be referenced by any skin style are kept in the skin so they can be references at a later time.
    • Remove On Save: If this option is selected, unreferenced images are removed from the skin the next time the skin file is modified and saved.
    • [Unused Images Count]: Displays the number of unreferenced images that are currently included in the skin file.

Test Cases

  • Verify that images used for any new skin styles do not get deleted when using the "Remove on Save" option. The mechanism used to delete images only checks specific styles, so if new style are added that reference images, we must account for those.
  • Verify that images associated with custom buttons (Toolbar Items) do not get deleted when "Remove on Save" is enabled.

Skin Editor (HTML5 Tripane): Size Tab

UI Controls

  • Use Browser Default Size: This option is checked by default and will use the default size of your web browser. You may deselect this option and set a manual size.
    • Top: adjust distance from the top of the screen
    • Bottom: adjust distance from the bottom of the screen.
    • Left: adjust distance from the left of the screen.
    • Right: adjust distance from the right of the screen.
    • Width: adjust the width of the skin
    • Height: adjust the height of the skin
  • Preview Full Size: Click this to open the Preview Window Size dialog. You are able to edit the size of the skin in the Preview Window Size dialog.
  • Anchors: anchor the position so resizing the skin in output will not override the manual size settings.

Skin Editor (HTML5 Tripane): Setup Tab

UI Controls

  • Navigation Pane:
    • Pane Position: Dropdown to select the position of the Webhelp pane. i.e. Left right, top, bottom.
    • Pane Size: Select the size of the pane in px.
    • Hide Navigation Pane on startup: Select this option to hide the navigation pane on startup.
  • Browser Settings: Selected by default. Deselect to manually choose browser settings i.e. toolbar, menu, location bar, status bar, resizable.

    • Note: In order for browser setting to take effect it must be launched via another browser window using either a JavaScript or URL.

  • Topic Settings:
    • Show navigation link at the top of topic: Select to show the navigation link at the top of the topic.
    • Show navigation link at the bottom of topic: Select to show the navigation link at the bottom of the topic.
  • Responsive Output Settings:
    • Enable responsive output: Select to enable responsive output.
    • Tablet Max-Width: Enter max width values for tablet medium.
    • Mobile Max-Width: Enter max width values for mobile medium.
