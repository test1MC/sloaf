UI Controls

  • Navigation Pane:
    • Pane Position: Choose from Right or Left. This is the location of the side menu when viewing in tablet/mobile view.
    • Fixed Header: Choose from None, Web, Tablet & Mobile, Mobile, or All. This will stick you top navigation so that it's always displayed even when you scroll down.
    • Slide-Out Menu Style: Choose from Drilldown (default) or Tree.  Tree is similar to Tripane HTML5 and the TOC editor where you expand books and can see all books and topics at a time.
    • Top Menu Levels to Show (Depth): Select how many levels of TOC items to include in top navigation menu.
    • Logo Url: Select the topic to be linked to the logo in the output. By default it will be your startup topic.
      • Browse: Opens Select Topic dialog.
