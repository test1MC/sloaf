Topic Toolbar

Bug Base Product Area:

  • Topic Toolbar

 

Description:

A feature for Web outputs (WebHelp, WebHelp Plus, WebHelp AIR, DotNet Help and HTML Help) which lets you inject a "WebHelp" toolbar into a topic. You do this by using a Topic Toolbar Proxy either in a Master Page or in a topic. The buttons which appear in the generated toolbar can be customized either in the project skin or locally (in the Master Page or topic). Any local settings applied to a Topic Toolbar Proxy should override any skin settings.
