Browse Sequences

Bug Base Product Area:

  • Browse Sequences

 

Description:

For online output, if you have several topics that you think end users should read in order, you can create browse sequences. When users view the compiled online output, they can use your browse sequences to "browse" through topics in a particular order.

 

This feature is supported in DotNet Help, HTML Help, WebHelp, WebHelp Plus, WebHelp AIR, and WebHelp Mobile output.
