You will need to enable WebDAV from Control Panel > Programs and Features > Turn Windows Features on or off. > Expand Internet Information Services, then World Wide Web Services, then Common HTTP Features. > Select WebDAV Publishing, and then click OK.

You may need to enable Windows and Basic authentication from Control Panel > Programs and Features > Turn Windows Features on or off. > Expand Internet Information Services, then World Wide Web Services, then Security > Select Basic Authentication and Windows Authentication, and then click OK. (close/reopen IIS Manager)

In IIS Manager, Select the Website > Open Authentication > Enable Windows Authentication

 

When setting up WebDAV, make sure the app pool being used for the site is 2.0 (.net 2.0)

 

Note for WebDAV:

In IIS Manager, select the Website > WebDAV Authoring Rules. In the Actions column (on the right), select Add Authoring Rule. Select Allow access to this content to: All users and Permissions: Read, Source, Write and then Click OK.
