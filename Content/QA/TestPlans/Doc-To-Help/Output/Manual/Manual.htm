Manual and PDF

Testing Notes:

Localization option is not used for Manual targets. Documentation does hot have this information.

 

To test the Fix Numbered Lists: option under advanced,

- create a new project and add a new document using C1H_prnomargin.dot template.

- Add a list of text and apply C1H Number special formatting to the text.

- Add a second document and add another list of text and apply the same C1H Number special formatting to the text.

- Build the target.

- Fix numbered lists: "Always" will reset the numbered list across pages.

- Fix numbered lists: "Never" will continue the numbered list across pages.

 

Checklist:

  • Test your outputs in Word and in the generated PDF
