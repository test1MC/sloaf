D2H_FANCY.DOTX

 

This source template has a page layout similar to D2H_NOMARGIN.DOTX, but it has a fancier styles that use color and a different font. When used with the complimentary target template, it will have a fancy design.

 Fancy 8.5" x 11" Source Template

D2H_PRFANCY.DOTX

 

This is the target template that is intended to accompany the D2H_FANCY.DOTX source template. It includes a page layout similar to D2H_PRNOMARGIN.DOTX, but it has a fancier design. This includes color, an image placed behind the text on the title page, an image placed behind the text in the headers, and a decorative image in the footers.

 Fancy 8.5" x 11" Target Template

D2H_FANCY_A4.DOTX

 

This is the source template for A4-sized paper.

 Fancy A4 Source Template

D2H_PRFANCY_A4.DOTX

 

This is the target template used to format the printed manual target (A4-sized paper).

 Fancy A4 Target Template

D2H_FANCY_POLICIES.DOTX

 

This source template is used to create policies & procedures documents. It has sample text and headings that you can use as a starting point for these manuals. It is based on D2H_FANCY.DOTX and uses the same styles and page layout as in that template. By default, the headers in this template are numbered.

 Policies and Procedures 8.5" x 11" Fancy Source Template

D2H_PRFANCY_POLICIES.DOTX

 

This is the target template that is intended to accompany the D2H_FANCY_POLICIES.DOTX source template. Like the PR_FANCY.DOTX target template, this template includes color, an image placed behind the text on the title page, an image placed behind the text in the headers, and a decorative image in the footers.

This is the Doc-To-Help default source template. It will be used unless you use your own or choose an alternate from this list.
