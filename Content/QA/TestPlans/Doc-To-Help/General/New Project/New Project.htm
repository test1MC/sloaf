Testing Notes:

New behavior for v5 when creating a new project based on existing file: See notes in Bug 98563:Ability to create a new project based on importing existing documents is missing from Create New Project wizard

From Andrey:

The problem is we must add into a new project at least one Word document to open it in Word, otherwise Word don't know about the new project.

So now:

- if some Word documents have been imported we open the first one in Word.

- if you cancel Import wizard or don't import any Word document, we suggest to add a default Word document and open it,

- if you cancel adding a default Word document we add a glossary Word document without prompt and open it.
