Attention Developers

1. Go here to find the list of bugs: !MCC Pending Dev ---Query shows all bugs currently in the four Bug Triage Iterations, an example is: "SOURCE\MadCloud Alpha\MadCloud - Bug Triage - 0-Now"

2. Grab an unassigned bug (can be Pending Dev or Bug Triage) ---Go by severity, then Backlog Priority.

3. Once you take it, move it into the current iteration (36), assign your name to it and add at least one task with an estimate (in hours) so we can gauge LOE.

Note: You can view all iterations here

To onboard yourself as a user:

  1. Go to: http://madcloudportal-stage.azurewebsites.net
  2. Use our master account credentials to create a new license (see below*)
  3. In the Add New License UI > Enter new license name, a vanity URL, and add your email to the Email field > Create
  4. Log out
  5. Log back in as yourself
