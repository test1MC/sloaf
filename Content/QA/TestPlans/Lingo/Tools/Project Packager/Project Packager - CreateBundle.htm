Statistics for the new bundle dialog:

 

 

Test Cases:

  • Title is correct
  • Spelling and Grammar are correct
  • No duplicate accelerators
  • All accelerators function
  • Clicking the icon in the upper left corner displays options:
    • Restore, Min., are disabled
    • Move, Size, Max., Close – All work
  • Close and <Enter> close dialog with changes
  • Dialog is resizable and redraws correctly
  • Right-clicking anywhere “Reset Persistent Data”
  • Dialog remembers setting and sizes from previous session
  • Dialog opens correctly on a dual monitor system
  • Dialog should show word and segment count (Make sure to use a smaller project and test that the counts match with the ones from the project)
  • Clicking "Close" should close the dialog and show the newly created ZIP file on disk:
