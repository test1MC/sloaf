New Project UI

  • !File > New Project

  • Shortcut keys Ctrl + Shift + N
  • Clicking New Project from the Start page - Under Getting Started.

Start New Project Wizard-Enter name and folder for the translated project...

 

 

Controls:

 

  • Name: Default name is NewProject<incremented #>
  • Folder: Default path is (C:\Users\UrUser\Documents\My Translated Projects). Lingo creates the My Translated Projects folder when you install the program.) All subfolders and files related to the translated project will be placed in this folder as you work on the project. If you want to have the project files placed in a different folder, use the "Browse" button.
  • Domain: You can type a subject category for your project, or select one from the drop-down. If you enter your own custom category in the Domain field, it will appear in the drop-down the next time you create a new project. Domain metadata appears throughout the project, such as when you select a suggestion in the translation memory.
  • Client:You can type the name of the client you are creating a translation project for. When you enter a client name in the field, it will appear in the Client drop-down the next time you create a new project. Client metadata appears throughout the project, such as when you select a suggestion in the translation memory.
  • Source language: Select the original language that is used in the project you are translating.
  • Target language: Select the language(s) that you want to use for the translation and click the right arrow to move it to the Selected Languages grid. If you need to remove a language from the Selected Languages grid, click the left arrow.
  • Bind to source control: This check-box will show the bind project to source control dialog when clicking next from the current dialog.
  • Back: This button is disabled in this window.
  • Next: Opens the next page of the wizard.
  • Finish:This button is disabled in this window.
  • Cancel: Closes the wizard.

  • Clicking ’Browse’ invokes the Browse For Folder dialog window.

Open Project

 

Opening the Open Project Dialog:

  • !File > Open
  • Press Ctrl + o keyboard shortcut
  • Start page > Open under Getting Started
  • Double-clicking the *.liprj, or *.liprjzip files in Windows Explorer

These will invoke the standard Windows Open Files Dialog with the filter set for *.liprj. File filter dropdown should list both *.liprj and *.liprjzip
