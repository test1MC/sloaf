Source Control - Add from Source Control

In the new project wizard, we added an option to import file(s) /project from source control since Lingo 10. 5991. List of available Source control servers can be found here.

Start new project wizard - Select Source Control Provider

Controls:

  • Next: Takes you to the browse for the source files dialog
  • Back: Take you to the Select files or project to translate dialog
  • Finish: Should be grayed out here.
  • Providers drop menu: Drop menu control that lists the 4 supported SC providers

    Changing the provider should change the section below it to show description and fields corresponding to the selected SC provider
