Source Control - Bind Lingo projects to Source Control

Lingo projects can be bound to SC in two ways: 

  • Upon project creation: by checking the Bind to source control check-box in the Enter name and folder for the translated project dialog of the Start new project wizard
  • From existing project project properties /Source Control Tab (!File > Project Properties > Source Control > Bind Project)
  • From Source Control Explorer/ Settings drop down for an existing project
